# 🎮 Rock Paper Scissors – Python Game

## 📌 Project Overview

This is a simple command-line Rock Paper Scissors game built using Python.
The user plays against the computer, and the winner is decided based on classic game rules.

## 🚀 Features

* User vs Computer gameplay
* Random computer move generation
* Input validation
* Clear win/lose/draw logic
* Beginner-friendly structure

## 🛠️ Tech Stack

* Python 3
* Random module

## 📂 Project Structure

Rock_Paper_Scissor/
│── main.py
│── README.md

## ▶️ How to Run

1. Clone the repository:

git clone https://github.com/sardeshballa/python_RockPaperScissor.git

2. Navigate to the project folder:

cd Rock_Paper_Scissor

3. Run the program:

python main.py

## 🧠 Game Logic

* Rock beats Scissors
* Scissors beats Paper
* Paper beats Rock
* Same choices result in a tie

## 📸 Sample Output

Enter your move = Rock, Paper or Scissor: Rock
Computer chose: Paper
Paper covers Rock, Computer wins!
